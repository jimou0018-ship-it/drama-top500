# 影视综内容 Top500 · 昨日热点看板

自动从 DA 数据平台抓取四大视频平台（抖音/B站/快手/小红书）影视综内容 Top500 榜单，
按「新增点赞量」取各平台 T-1 日 Top20 合并展示。

🔗 **在线查看**：<https://jimou0018-ship-it.github.io/drama-top500/>

## 功能
- T-1 日新增点赞 Top20（各平台）合并表
- 历史热点汇总（折叠展开）
- 多维筛选（平台、日期、内容分组）+ 关键词搜索
- 任意列排序
- 一键导出 Excel
